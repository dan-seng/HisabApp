💰 About Hisab

Smart finance automation that works for you, not against you

What is Hisab?

Hisab is a smart offline-first finance app that automatically tracks bank transactions using SMS messages. No manual entry required.

• Offline First • Automatic Tracking

How it works

The app scans SMS notifications from banks, extracts transaction details, and updates balances automatically in real-time.

• SMS Scanning • Real-time Updates

Unified Dashboard

All bank accounts and balances are shown in one clean, simple financial dashboard that gives you complete control.

• Multi-bank Support • Clean Interface

Privacy First

All data stays on the device. No cloud, no tracking, and fully offline storage. Your financial data is yours alone.

• 100% Private • Local Storage

class PrivacyManager {
    private val encryptionKey = generateKey()
    
    fun encryptData(data: String): String {
        // Your data never leaves your phone
        return AES.encrypt(
            data, 
            encryptionKey, 
            getLocalIV()
        )
    }
    
    fun isLocalOnly(): Boolean = true
}
